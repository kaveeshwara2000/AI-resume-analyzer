## AI Resume Analyzer Web Application
Modern web application for an AI-powered Resume Analyzer. This application consists of a React frontend and a Spring Boot backend, communicating via REST APIs.
- Frontend: React (Vite-based), Tailwind CSS (sleek dark/light modes, custom UI), Axios (HTTP communication), React Router DOM (client-side routing), Lucide React (icons).
- Backend: Spring Boot, Spring Security (JWT authentication), Spring Data JPA (Hibernate), MySQL, Apache PDFBox (PDF parsing), and Google Gemini API (AI analysis).
